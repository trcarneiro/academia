# Quickstart: UX Standardization

## Prerequisites
- Node.js 18+
- NPM

## Running the Project
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Access the application at `http://localhost:3000`.

## Verifying UX Changes
1. Navigate to the **Courses** module.
2. Verify that the layout matches the **Instructors** module (Premium Standard).
3. Check for:
   - Gradient headers
   - Consistent colors (`#667eea`, `#764ba2`)
   - Responsive layout
   - No modals
